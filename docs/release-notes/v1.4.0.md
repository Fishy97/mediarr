# Mediarr 1.4.0 Release Notes

Mediarr 1.4.0 makes media-server ingestion automatic. After connecting Jellyfin, Plex, or Emby, users no longer need to remember to press Sync before Mediarr can build activity-aware recommendations.

## Highlights

- Auto-sync is enabled by default for configured media-server integrations.
- Saving a valid server URL and API key/token immediately queues the first sync.
- Startup recovery queues syncs for configured integrations that have never run or are stale.
- A backend scheduler keeps integrations fresh on the configured interval.
- The default interval is 6 hours.
- The Integrations UI now exposes auto-sync enablement, interval, and next-sync status.
- The scheduler avoids duplicate jobs when a sync is already queued or running.
- Plex automatic sync continues using the watch-history cursor.

## Safety Notes

Automatic ingestion is read-only. It imports inventory/activity, refreshes advisory recommendations, and records audit events. It does not delete, move, or modify media.
